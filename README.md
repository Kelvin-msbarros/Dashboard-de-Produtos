# Dashboard-de-Produtos
Desenvolvimento de dashboard para análise de desempenho comercial, apresentando indicadores de faturamento por categoria, quantidade e faturamento de vendas por estado, além do cálculo de ticket médio. Os dados foram tratados e estruturados utilizando SQL, permitindo análises estratégicas e apoio à tomada de decisão.
